# terminal-ui
# 💻 Interactive Web Terminal UI
A sleek, lightweight, custom-styled web terminal interface built using pure HTML, CSS, and Vanilla JavaScript.

![Tech Stack](https://img.shields.io/badge/tech-HTML5%20%7C%20CSS3%20%7C%20JS-orange)
---
## ✨ Features
* **Interactive CLI Interface:** Real-time keypress capture with custom CSS blinking cursor.
* **Built-in Commands:**
  * `help` — List available commands.
  * `whoami` — Displays current active user profile.
  * `date` — Returns system date and time.
  * `echo [text]` — Prints input text back to screen.
  * `clear` — Wipes the terminal screen history.
* **Toolbar Actions:**
  * Red Window Button (Exit) — Closes active terminal session.
  * Plus Tab Button (`+`) — Resets window session.
* **Zero Dependencies:** Pure Vanilla JavaScript—no external frameworks required.
